# AI LoL Coach — встановлення та ключі

Покрокова інструкція. Загалом ~10 хвилин, усі ключі безкоштовні, картка не потрібна.

---

## 1. Встанови програму

Запусти **`AI LoL Coach Setup 0.1.0.exe`** → далі → готово.
Встановлюється тільки для тебе, права адміністратора не потрібні.
Запускати: ярлик **AI LoL Coach** у меню Пуск.

Після запуску програма створить файл налаштувань і **покаже банер**, у якому
написано, яких ключів бракує і де саме лежить файл.

---

## 2. Відкрий файл налаштувань

Натисни **Win+R**, встав і натисни Enter:

```
%APPDATA%\lol-coach
```

У теці, що відкриється, відкрий файл **`.env`** через Блокнот
(права кнопка → Відкрити за допомогою → Блокнот).

---

## 3. Riot-ключ — обовʼязковий

Без нього програма не бачить твої матчі, ранг і ротацію героїв.

1. Відкрий **https://developer.riotgames.com** і увійди своїм Riot-акаунтом.
2. Знайди блок **DEVELOPMENT API KEY**.
3. Постав галочку «Я не робот» і натисни **REGENERATE API KEY**.
4. Скопіюй ключ (починається на `RGAPI-`) і встав у `.env`:

```env
RIOT_API_KEY=RGAPI-твій-ключ
```

> ⚠️ **Найважливіше:** цей ключ живе лише **24 години**. Щодня доведеться
> заходити на сайт, тиснути Regenerate і вставляти новий.
> Щоб позбутися цього — подай заявку на безкоштовний **Personal API Key**
> (на тому ж сайті: Register Product → Personal). Він не протермінюється.

---

## 4. Groq-ключ — для AI-порад

Безкоштовно, ~1000 запитів на день. Це основний «мозок» коуча.

1. Відкрий **https://console.groq.com**, увійди (можна через Google).
2. Ліворуч у меню → **API Keys** → **Create API Key**.
3. Скопіюй одразу (показується лише раз!) і встав:

```env
GROQ_API_KEY=gsk_твій-ключ
```

---

## 5. Gemini-ключ — необовʼязковий

Додає дві речі: AI **дивиться на твій екран** у грі (мінімапа, позиції ворогів)
і підтягує актуальні білди з інтернету.

1. Відкрий **https://aistudio.google.com/apikey** → **Create API key**.
2. Встав:

```env
GEMINI_API_KEY=AIza-твій-ключ
```

---

## 6. Збережи файл і перезапусти програму

Ключі зчитуються лише під час запуску. Закрий програму
(**правою кнопкою по іконці в треї → Quit**) і запусти знову.

Якщо все правильно — банер зникне, а на головній зʼявляться патч і герої
безкоштовної ротації.

---

## Що працює **без жодного ключа**

Віджет у грі стартує одразу: KDA, фарм за хвилину, віжн, золото, таймери
дракона й барона та підказки. Це працює через **локальний** API самої гри,
без інтернету й без ключів.

Ключі потрібні лише для: аналізу 20 ігор, AI-порад, новин і ротації героїв.

---

## Як користуватись

- **Головне вікно** — впиши `Нік#ТЕГ`, обери регіон, натисни **Аналіз**.
  Зірочка ★ зберігає акаунт, щоб не вводити щоразу.
- **Віджет** — зʼявляється сам, коли починається гра.
  ⚙ — прозорість, розмір, які панелі показувати. 📌 — закріпити на місці.
- **Іконка в треї** — клік показує/ховає віджет, права кнопка — меню.
  Закриття вікна **не вимикає** програму, вона згортається в трей.
  Щоб вийти повністю — **Quit** у меню трею.
- **Гарячі клавіші** — `Ctrl+Shift+H` показати/сховати, `Ctrl+Shift+X` клікати «крізь» віджет.

> 🖥 **Важливо:** щоб віджет було видно поверх гри, у League має бути
> **Borderless** (Esc → Options → Video → Window Mode → Borderless).
> У режимі Exclusive Fullscreen Windows ховає будь-які вікна поверх гри —
> це обмеження системи, а не помилка програми.

---

## Приватність

- Знімається **тільки вікно League**, ніколи не весь екран. Якщо вікна гри
  нема — не надсилається нічого.
- Кадр іде в AI щонайбільше раз на хвилину, лише під час гри і лише якщо
  картинка справді змінилась.
- Локальний сервер слухає тільки `127.0.0.1` — ніхто з твоєї мережі до нього
  не достукається.
- Ключі лежать лише у твоєму `.env` і не вшиті в інсталятор.
- Тільки офіційні API Riot, лише читання. Програма **радить**, а не грає за тебе.
