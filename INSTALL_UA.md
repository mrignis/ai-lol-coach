# AI LoL Coach — Встановлення

**Ключі не потрібні.** Постав і грай — аналіз, AI-коучинг, новини й віджет у грі
працюють одразу (усе йде через сервер, який тримає ключі за тебе).

## Встановлення

1. Запусти **`AI LoL Coach Setup.exe`** → Далі → Готово.
   Ставиться тільки для тебе, права адміністратора не потрібні.
2. Запусти **AI LoL Coach** з меню Пуск (або ярлик на робочому столі).

І все. Впиши свій Riot ID (`Нік#ТЕГ`), обери регіон, натисни **Аналіз**.

## Як користуватись

- **Головна** — аналіз останніх 20 ранкед-ігор; зірочка ★ зберігає акаунт.
- **Білди** — актуальний білд і план на матчап, коли ти вже в грі.
- **Віджет у грі** — з'являється сам, коли починається матч. Показує KDA, CS/хв,
  віжн, таймери об'єктів і AI-підказки. Пересувається будь-куди, 📌 закріплює,
  ⚙ — прозорість/розмір і які панелі показувати.
- **Іконка в треї** — клік показує/ховає віджет, права кнопка — меню.
  Закриття вікна ховає програму в трей, вона працює далі. Вийти повністю —
  **Quit** у меню трею.
- **Гарячі клавіші** — `Ctrl+Shift+H` показати/сховати, `Ctrl+Shift+X` клікати «крізь».

> 🖥 Щоб віджет було видно поверх гри, постав League у **Borderless**
> (Esc → Options → Video → Window Mode → Borderless). У повному екрані
> (Exclusive Fullscreen) Windows ховає будь-які оверлеї — це обмеження системи.

## Що де працює

- **Віджет у грі** читає локальний API League (`127.0.0.1:2999`) напряму —
  без інтернету й ключів, з ПК нічого не виходить, окрім (за бажанням)
  скріншотів для AI-підказок.
- **Аналіз / AI / новини** йдуть через сервер, тож ти не маєш справи з ключами.

## Приватність

- Знімок екрана (для AI-підказок) бере **тільки вікно League**, ніколи не весь
  екран, щонайбільше раз на хвилину і лише під час гри з відкритим віджетом.
- Локальний сервер слухає тільки `127.0.0.1` — ніхто з мережі не достукається.

## Свій бекенд (необов'язково)

Не хочеш залежати від спільного сервера? Можеш підключити власні ключі — див.
`.env.example` і `README.md`. Для звичайного використання не потрібно.
